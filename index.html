🤖 AI Settings — API Key

Required for Tier Finder and AI pricing generator
Stored in this browser only (localStorage). Not sent anywhere except Anthropic.
How to get your API key
1. Go to console.anthropic.com → sign in or create a free account
2. Click API Keys in the left sidebar → Create Key
3. Copy the key (starts with sk-ant-) and paste it above
4. Recommended: set a monthly spend limit (e.g. $10 AUD) under Billing → Limits so there are no surprises
5. Each team member enters the key once on their own device — it's saved in their browser
💡 Cost guide: The Tier Finder uses ~1,000–2,000 tokens per analysis (≈ $0.003–0.006 AUD). The AI pricing generator uses ~300 tokens (≈ $0.001). Normal usage costs cents per month, not dollars.

Supports: .xlsx, .xls, .csv · Max 10MB
Expected format: columns for Item Name, Quantity (and optionally Price/unit)
Supported file formats:
Excel (.xlsx/.xls): The tool reads the first sheet. Columns detected automatically — any column containing "item", "product", "description" is used as the item name. Columns containing "qty", "pieces", "count", "quantity" are used as quantities.
CSV: Same column detection. First row treated as headers.
Your "Sales by Product" report: The tool recognises the hierarchical format (Store → Category → Item) and imports item names + piece counts automatically.
